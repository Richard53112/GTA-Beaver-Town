# GTA-The-River-City---System
รายละเอียดทั้งหมดของสคริปต์ GTA The River City

✅ = ทำแล้ว
❌ = ยังไม่ได้ทำ
( อัพเดทล่าสุด 14.11.2565 )


• หมวดหมู่ : ระบบบัญชีผู้ใช้งาน
✅ สมัครสมาชิก
✅ เข้าสู่ระบบ
✅ ถ้าไม่เข้าสู่ระบบภายใน 60 วินาทีจะถูก Kick
✅ ถ้าใส่รหัสผ่านผิดเกิน 3 ครั้งจะถูก Kick
❌ เงินเขียว
❌ เงินในธนาคาร
❌ เงินแดง


• หมวดหมู่ : ระบบสำหรับผู้เล่น
✅ สร้างตัวละคร ( กำหนดเพศ , อายุ , สกินตัวละคร ) - เป็นรูปแบบ TextDraw เหมือนสคริปต์ดั้งเดิมของพี่ JustinzDev
✅ แพลนมุมกล้องจากท้องฟ้าสู่พื้นดินที่เป็นจุดเกิดสำหรับผู้เล่นใหม่ , หลังจากกด Confirm การสร้างตัวละครแล้ว
✅ ระบบตรวจสอบอุปกรณ์ว่าเข้าเล่นผ่าน "คอมพิวเตอร์" หรือ "มือถือ" หลังจากผู้เล่นเข้าสู่ระบบ


• หมวดหมู่ : ระบบกระเป๋าเก็บของผู้เล่น
✅ น้ำหนักกระเป๋าสำหรับผู้เล่น ( เริ่มต้นที่ 30.0 KG )
✅ หน้าจอสำหรับระบบกระเป๋าเก็บของส่วนตัวแบบผู้เล่นที่เป็น TextDraw เหมือนกับสคริปต์ดั้งเดิมของพี่ JustinzDev
✅ ระบบแจ้งเตือนตอนผู้เล่นได้รับ Item หรือลบ Item ออกจากกระเป๋าเก็บของ
❌ สามารถกด 'ใช้งานไอเท็ม, ให้ผู้เล่น, ทิ้งไอเท็ม, รายละเอียดไอเท็ม, เพิ่มไปช่องคีย์ลัด' ได้
❌ ระบบ 'ช่องคีย์ลัด' สำหรับเพิ่มไอเทมเพื่ออำนวยความสะดวก ( กดปุ่ม H เพื่อใช้งานช่องเก็บไอเทมคีย์ลัด )
❌ รายละเอียดเกี่ยวกับไอเทมทั้งหมดในเซิร์ฟเวอร์
❌ เพิ่มผลลัพธ์หลังจากผู้เล่นกดปุ่ม 'ใช้งานไอเท็ม' หลังจากคลิกเลือกไอเทมในกระเป๋าเก็บของ
❌ เพิ่มท่าทางตอนผู้เล่นเปิดกระเป๋า ( จะเป็นท่าทางก้มเก็บกระเป๋า ) จนกว่าผู้เล่นจะปิดกระเป๋าจะกลับมาท่าทางเดิมตามปกติ


• หมวดหมู่ : ฝั่ง Server
✅ เพิ่ม TextDraw โลโก้ของเซิร์ฟเวอร์ไว้ที่มุมขวาล่างของหน้าจอ
❌ เมนูสำหรับ 'อำนวยความสะดวก' ให้กับผู้เล่นตอนกดปุ่ม Y ใช้งานกระเป๋า ( พวก Status, Give  Money, Ticket, GPS ประมาณนี้ )
✅ ระบบ GPS นำทาง


• หมวดหมู่ : ระบบสำหรับ Admin

❌ คำสั่งทั่วไปสำหรับ Admin
❌ ระบบช่องพูดคุยระหว่าง Admin (/a)
❌ คำสั่งประกาศข้อความสำหรับ Admin
❌ ระบบ Ban
❌ ระบบ Kick
❌ ระบบ Jail / Unjail
