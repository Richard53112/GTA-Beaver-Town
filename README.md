• หมวดหมู่ : ระบบกระเป๋าเก็บของผู้เล่น
✅ น้ำหนักกระเป๋าสำหรับผู้เล่น ( เริ่มต้นที่ 30.0 KG )
✅ หน้าจอสำหรับระบบกระเป๋าเก็บของส่วนตัวแบบผู้เล่นที่เป็น TextDraw เหมือนกับสคริปต์ดั้งเดิมของพี่ JustinzDev
✅ ระบบแจ้งเตือนตอนผู้เล่นได้รับ Item หรือลบ Item ออกจากกระเป๋าเก็บของ
✅ สามารถกด 'ใช้งานไอเท็ม, ให้ผู้เล่น, ทิ้งไอเท็ม, รายละเอียดไอเท็ม, เพิ่มไปช่องคีย์ลัด' ได้
✅ ระบบ 'ช่องคีย์ลัด' สำหรับเพิ่มไอเทมเพื่ออำนวยความสะดวก ( กดปุ่ม H เพื่อใช้งานช่องเก็บไอเทมคีย์ลัด )
✅ รายละเอียดเกี่ยวกับไอเทมทั้งหมดในเซิร์ฟเวอร์
✅ เพิ่มผลลัพธ์หลังจากผู้เล่นกดปุ่ม 'ใช้งานไอเท็ม' หลังจากคลิกเลือกไอเทมในกระเป๋าเก็บของ
❌ ตรวจสอบระบบ "เปลี่ยนหน้า" กระเป๋าเก็บของว่าทำงานปกติหรือไม่?
✅ เพิ่มท่าทางตอนผู้เล่นเปิดกระเป๋า ( จะเป็นท่าทางก้มเก็บกระเป๋า ) จนกว่าผู้เล่นจะปิดกระเป๋าจะกลับมาท่าทางเดิมตามปกติ
